# Precision Tech Studio - 官网设计简案

## 设计理念

### 核心主题：**信任与精准**
- **视觉语言**：现代极简 + 科技感，强调"高确定性交付"的承诺
- **色彩方案**：深蓝 (#1e3a8a) + 白色 + 浅灰 (#f3f4f6)，辅以翠绿 (#10b981) 作为强调色
- **排版**：Geist Sans (现代、专业) + Georgia (优雅的衬线，用于标题)
- **风格**：B2B SaaS 风格，但不失温度感

## 页面结构

### 1. **Hero Section**
- 标题：「高确定性交付 × AI 赋能」
- 副标题：为澳洲中小企业提供精品软件解决方案
- CTA：「获取咨询」+ 「查看案例」
- 背景：深蓝渐变 + 几何图案

### 2. **核心服务**
- **AI 业务流程自动化**：2-4周快速交付，可量化ROI
- **低代码 + 高级定制**：成本降低50%，交付速度提升2倍
- **遗留系统现代化**：风险管理，平稳过渡

### 3. **为什么选择我们**
- 10年开发经验 + 5年项目管理
- 澳洲本地，深刻理解隐私法与数据主权
- 透明化协作，实时项目管理

### 4. **成功案例**（3-4个案例）
- 简短的故事 + 数字化成果

### 5. **定价**（简化版）
- 三个档位：Starter, Professional, Enterprise

### 6. **团队**
- 创始人介绍 + 价值观

### 7. **联系方式**
- 表单 + 社交链接

## 设计元素

- **图标**：Lucide React 图标库
- **动画**：Framer Motion，微妙的进入动画和悬停效果
- **卡片**：阴影 + 边框，提升视觉层次
- **排版层次**：H1 (48px) → H2 (36px) → H3 (24px) → Body (16px)
